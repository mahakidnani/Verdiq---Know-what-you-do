# Verdiq

Verdiq is a cloud-based stock analysis platform designed for first-time investors who find traditional financial tools overwhelming.

Instead of displaying complex ratios and raw data, Verdiq translates financial metrics into simple, human-readable insights and provides a clear verdict on whether a stock appears overvalued, undervalued, or fairly valued.

---

##  Problem

Most stock analysis platforms:
- Show complex metrics like P/E, ROE, EBITDA without explanation
- Expect users to interpret financial data themselves
- Provide charts without context or storytelling
- Are not designed for beginner investors
- Do not answer the key question: *“Should I invest?”*

---

## Solution

Verdiq simplifies stock analysis by focusing on clarity and decision-making.

### Key Features:
-  **Plain English Insights**  
  Converts financial data into easy-to-understand explanations

-  **Narrative Charts**  
  Charts include contextual captions explaining trends and risks

-  **Verdiq Score**  
  A simplified scoring system to evaluate stock health

-  **Clear Verdict**  
  Instantly shows whether a stock is:
  - Undervalued
  - Fairly Valued
  - Overvalued

- **Question-Based UI**  
  Replaces jargon with user-focused questions like:  
  *“Is this stock expensive right now?”*



##  Target Users

- Beginner investors
- Students and young professionals
- Users new to platforms like Groww or Zerodha



##  Tech Stack (Planned)

- Frontend: React.js
- Backend: Node.js / Express
- Database: MongoDB / PostgreSQL
- Cloud - Supabase
- APIs: Financial data APIs
- AI Layer (optional): LLM for generating insights



##  Project Scope

This is a prototype built as part of a personal learning project.  
It is intended for educational purposes and does not provide financial advice.



##  Disclaimer

To make stock investing understandable, accessible, and actionable for everyone — not just finance experts.
